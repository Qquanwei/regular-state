API清单

API清单

API根据知识结构散落在各个章节，不方便查找，本章提供了快速索引清单

注意

※ 标记的含义是此Manager实例方法与Stateman的一致(Manager为Stateman的子类), 所以部分函数会直接链接到Stateman文档_

API列表

  • restate: 实例一个Manager对象
    • return Manager
  • manager.start: 启动路由
  • manager.state: 注册路由
  • ※manager.go: 根据别名跳转
  • ※manager.nav: 根据url跳转
  • ※manager.on: 绑定事件
  • ※manager.off: 解绑事件
  • ※manager.emit: 手动触发事件
  • 指令
    • r-link: 跳转链接便利生成指令
    • r-view: 标志子模块的插入位
  • 内建事件
    • begin: 跳转开始前触发，你可以组织这次跳转
    • end: 跳转结束触发
    • notfound: 跳转目标未找到触发
  • 跳转参数option
    • option.param: 路由参数
    • option.current: 目标路由节点
    • option.previous: 跳转前路由节点
  • ※manager.is: 判断当前URL是否满足传入的stateName
  • ※manager.encode: 根据状态名和参数编码URL
  • ※manager.decode:从URL解码状态和参数
  • ※manager.stop: 停止路由
